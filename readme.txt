SUMMARY

DAILY PROCESS:
- Click on the Daily Process button which will:
    + Create a backup of the file at the same directory where the opened 
    file is.
    + Import the files from the tradersync export into the TraderSync 
    Export sheet.
    + Processes the TDSync export into the Trade Log.

WEEKLY PROCESS:
- Click on the Weekly Process button which will:
    + Create a backup of the file at the same directory where the opened 
    file is.
    + Clear the weekly one pager.
    + Processes entries from the journal to the weekly one pager if they are 
    marked for processing in the Weekly One Pager column. 
    + Asigns unique TradeID



HOW TO MAINTAIN:
- Adding new Market tickers: 
    + Add a ticker to the MARKET_TICKERS list if you want any additional ticker 
    to have market treatment.
